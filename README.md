# ConfigurationLanguageToml
Данная версия приложения является предрелизной, так как в ней не реализованы тесты, но реализована вся основная логика приложения.

Были выполнены следующие изменения:
Реализован Visitor для преобразования AST
- Создан ConfigVisitor, наследующий ConfigGrammarBaseVisitor
- Реализованы методы Visit для всех правил грамматики
- Добавлено вычисление значений констант (EvaluateExpression)
- Обработка: констант, чисел, строк, словарей и вложенных выражений

Добавлен обработчик ошибок парсера
- Реализован ErrorHandler с интерфейсами IAntlrErrorListener<IToken> и IAntlrErrorListener<int>
- Отдельная обработка лексических и синтаксических ошибок
- Улучшенные сообщения с указанием строки и позиции
- Дополнительные проверки: заглавные буквы в именах и незакрытые комментарии

Добавлен конвертер из AST в TOML
- Реализован TomlConverter с поддержкой библиотеки Tomlyn
- Конвертация констант, словарей и вложенных структур
- Автоматическое вычисление значений выражений (EvaluateExpression)
- Форматирование вывода: одиночные и множественные таблицы
- Поддержка преобразования DictionaryExpression во вложенные TOML-таблицы

Была добавлена реализация класса Program.cs
