# ConfigurationLanguageToml
Приложение для конвертации в файл toml
Данная первая ветка была реализована для создания абстракции приложения. В ней были сделаны следующие изменения:
- Создан файл грамматики ConfigGrammar.g4 
- Определены правила парсера и лексера 
- Реализованы все требуемые конструкции языка 
- Добавлена поддержка двоичных чисел и словарей.
- Добавлены классы AST-модели для языка конфигураций
- Созданы базовые классы узлов AST (ConfigModel, ConstantDeclaration)
- Реализованы типы выражений: Constant, Number, String, Dictionary
- Добавлен DictionaryDeclaration с поддержкой KeyValuePair
- Модель готова для реализации Visitor
